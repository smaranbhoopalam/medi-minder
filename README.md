# Medi-Minder

AI powered medication reminder and stress monitoring system.

## Features
- Google authentication
- Medicine verification using camera
- Alert family if medicine is not taken
- Stress calculation using behavior data
- AI suggestions for stress reduction
- Graph visualization of health data

## Tech Stack
- Frontend: React
- Backend: Node.js
- AI: LLM integration
- Database: Firebase / MongoDB
